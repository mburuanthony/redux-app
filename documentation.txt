1. Run npm install or yarn install, to install dependencies for the project.

2. If you will be usimg the project in a git repository, add this the following to your .gitignore file:
    /node_modules
    /.pnp
    .pnp.js
    /coverage
    /build
    .DS_Store
    .env.local
    .env.development.local
    .env.test.local
    .env.production.local
    npm-debug.log*
    yarn-debug.log*
    yarn-error.log*

3. Run npm start to start the development server and view it in the browser.
